# SystemResourceMonitor

TODO 监视系统资源占用，揪出那个高占用的进程以及偷跑的进程

功能：

1. 程序启动后，开始监听所有进程
2. 用户通过`Ctrl+c`退出程序，输出报告，报告包括
   1. 每个进程的：起止时间，CPU、内存、IO、网络占用折线图
   2. 特别标注这些进程：
      1. 在监听范围内启动和结束的进程
      2. CPU占用TOP3进程（峰值）
      3. 内存占用TOP3进程（峰值）
      4. IO占用TOP3进程（峰值）
      5. 网络占用TOP3进程（峰值）

需要注意：

1. 多进程进程建议归并
2. 用户可以自定义扫描频率
3. 结果展示应该更直观
